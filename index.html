De ce CohesivX

Mai mult decât un indicator tehnic clasic

RSI și MACD îți arată forma prețului. CohesivX BTC API merge mai adânc: traduce structura internă a trendului Bitcoin și contextul macro global într-un set de indici normalizați și ușor de consumat în aplicații, dashboard-uri sau modele de risc.

IC_BTC & ICD_BTC
Structură + direcționalitate între 0–100

IC_BTC structural măsoară coerența trendului (raport între flux, preț și volatilitate). ICD_BTC direcțional măsoară cât de aliniate sunt mișcările zilnice cu trendul dominant (bull vs bear). Totul normalizat în 0–100, gata pentru scoring intern.

Regim coeziv
Regim de ciclu Bitcoin, explicit

Modelul derivă automat un regim de ciclu: bază, acumulare, bull structural, bull târziu / top, bear structural timpuriu, bear avansat, reechilibrare. Ușor de mapping în UI sau în modele de alocare dinamice.

Context global
Scor de risc macro agregat

Integrare directă cu SPX, DXY, Gold, VIX și Oil într-un macro risk score și un label simplu: risk-on, echilibrat sau risk-off. Bitcoin este citit în context, nu izolat.

FAQ

Întrebări frecvente

Este CohesivX BTC API un sistem de semnale de tranzacționare?
Nu. CohesivX BTC API este un indice de structură și regim, construit pentru evaluarea riscului și înțelegerea ciclului BTC. Poate fi folosit ca intrare în sisteme automate, dar nu reprezintă recomandare financiară și nu livrează „buy/sell” direct.
Ce timeframe folosește modelul?
Modelul curent lucrează pe timeframe daily (1D), cu o fereastră mobilă de aproximativ 260 de zile pentru calculul indicilor coezivi, al volatilității și al contextului macro. Integrarea pe alte timefram-uri poate fi discutată pentru planurile enterprise.
Ce surse de date sunt folosite?
Prețul BTC este agregat de pe surse de lichiditate majore (ex. Binance), iar contextul macro folosește indici standard (SPX, DXY, Gold, VIX, Oil). Modelul lucrează pe date de închidere (close) și nu expune direct feed-ul brut de preț, ci rezultatul procesării coezive.
Cum este actualizat indicele?
Pipeline-ul este automatizat: datele sunt preluate, normalizate și trecute prin modelul coeziv, apoi publicate în endpoint-urile API (în general o dată pe zi, după închiderea zilei). Pentru clienți enterprise, sunt posibile frecvențe de update custom.
Pot folosi CohesivX BTC API într-un produs comercial?
Da. Planurile Pro și Institutional sunt gândite pentru integrare în produse comerciale: dashboard-uri, platforme de analiză, roboți, aplicații mobile. Licența acoperă redistribuirea indicelui în UI-urile proprii, cu mențiunea sursei.
Modelul se poate extinde și la alte active?
Modelul coeziv este generalizabil. Versiunea actuală este calibrată pentru Bitcoin + context macro. Pentru integrări multi-asset (ETH, indici, FX), se pot defini proiecte custom în cadrul planului Institutional.
